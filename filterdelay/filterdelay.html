This is a download page for my filter and delay plugin.

Here is an audio demo. It uses JUCE, which is a c++ API for plugins and applications. The filters are state variable highpass and lowpass in series. There is 32x upsampling for the filters. The delay is a fractionally interpolated delay with sine wave modulation of the delay time. The modulation is 90 degrees out of phase between the left and right channel.
